founderstaff ships **75 skills** distributed across **10 packs**. Each skill is one of three types:

- **Chatmode** (`@name`) — a persistent persona you switch into
- **Prompt** (`/name`) — a slash command, one-shot, contracted output
- **Instructions** — always-on guardrails scoped by file glob

## How to choose

- **Just starting?** Install `core` only. Activate `chief-of-staff`. Add packs as needs surface.
- **Solo consultant?** `core + research + content + sales + delivery + compliance + finance`. See [AI Consultancy example](/examples/ai-consultancy/).
- **Indie SaaS builder?** `core + product + research + content + marketing + finance`. See [Indie SaaS example](/examples/indie-saas/).
- **Content creator?** `core + content + research + marketing`. See [Content Creator OS](/examples/content-creator/).
- **Local service business?** `core + sales + marketing + operations + finance`. See [Local Service Business](/examples/local-service/).
