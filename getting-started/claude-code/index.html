Get started with Claude Code

This walkthrough takes you from zero to “I have an AI chief of staff” in about 5 minutes. We’ll use Claude Code — Anthropic’s CLI agent — as the host.

Prerequisites

ToolMin versionInstall
Python3.10+python.org — python --version
Git2.30+git-scm.com — git --version
Claude Codelatestnpm install -g @anthropic-ai/claude-code then claude login

Note

founderstaff itself has zero Python dependencies — sync.py and relearn.py use only the standard library. No pip install, no virtualenv.

Step 1 — Create the workspace

  1. Make a folder for your new venture.

  2. Create the .platform/ directory.

  3. Copy the four scaffolding files from the founderstaff repo.

Step 2 — Configure tokens

Open .platform/platform.config.json and fill in your brand identity:

These get substituted into every installed skill file. BRAND appears in chatmodes. OPERATOR is your name (used by content + sales). EMPLOYER is your day-job employer name (used by the compliance firewall). Leave empty if not applicable.

Day-2 commands

CommandWhat it does
python .platform/relearn.pyPull latest upstream, re-sync (preserves local edits)
python .platform/relearn.py --statusShow installed skills + versions
python .platform/relearn.py --dry-runShow what would change
python .platform/relearn.py --pack n8nAdd a pack
python .platform/relearn.py --updateBump pinned versions to upstream latest
