Examples overview

Four real-world founder archetypes. Each walkthrough is a complete build: pack selection → roster → rhythms → week-in-the-life with actual commands and expected outputs.

AI Consultancy

Solo AI consultant running on top of a day job. Full compliance + delivery + finance + content engine. Read walkthrough →

Indie SaaS

Solo developer shipping a paid product. PRD → build → launch → growth loop. Read walkthrough →

Content Creator OS

Newsletter + YouTube + LinkedIn operator. Weekly batch + repurpose flow. Read walkthrough →

Local Service Business

Non-tech founder (clinic, agency, studio). SOPs + sales + cashflow visibility. Read walkthrough →

Reading these

Each example walkthrough has the same structure:

  1. The operator — who they are, what constrains them
  2. Pack selection — which packs and why
  3. Roster — which agents to activate
  4. Rhythms — the schedule
  5. A week in the life — actual commands by day, expected outputs
  6. Failure modes — what to do when something goes sideways

Use them as templates. Steal the manifest, steal the roster, steal the rhythm set. Customize the tokens and you’re 80% there.
