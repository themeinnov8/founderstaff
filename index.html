Why founderstaff

Most founders don't fail at ideas.
They fail at execution.

Decision fatigue. Scattered state. Solo blind spots. Compliance drift. founderstaff turns your editor into a chief-of-staff plus a research analyst plus a content creator plus a sales coach - all reading the same workspace, all guardrailed.

AI

Agentic, not chatty

Every chatmode declares its tools. Every prompt has a contract. Operators get one next action, not a buffet.

PK

Packs you opt into

Core is always on. Add research, content, sales, marketing, operations, delivery, compliance, finance, or product as you need them.

RY

Rhythms & councils

Schedule daily, weekly, or monthly invocations. Run multi-agent councils for cross-functional decisions.

CP

Compliance baked in

Pre-publish gate. Day-job firewall. Citation discipline. Decisions logged. Local edits respected.

HM

Selective hiring

The @hiring-manager refuses to forge new skills without a MECE-passing triage and operator confirmation. No skill bloat.

HS

One source, two hosts

Same skill files render to .github/ for Copilot or .claude/ for Claude Code. Pick your editor.

How it works

From idea to working OS in 4 steps

Clone the platform

One git clone + one python relearn.py. No package install. Stdlib only.

Pick your packs

Core is required. Add research, content, sales - whatever your venture needs.

Instantiate your roster

/instantiate-agent chief-of-staff. Activate the agents you actually want.

Set rhythms, run councils

Schedule daily standups. Convene a strategy council. Operate.

Claude Code recommended

Skills become .claude/agents/ (subagents) and .claude/commands/ (slash commands).

python relearn.py --target claude

Best for terminal-first operators. Anthropic's CLI agent.

GitHub Copilot

Skills become .github/chatmodes/ and .github/prompts/.

python relearn.py --target copilot

Best for VS Code-native workflows.

One command. Full team on staff.

founderstaff installs as Python stdlib. No deps, no virtualenv, no package manager. Just git pull + render.

  • Path translation: pack-first source -> flat .github/ or .claude/ targets.
  • Selective installs via packs + extra-skills + disabled-skills.
  • Token substitution: {{BRAND}}, {{OPERATOR}}, {{EMPLOYER}}.
  • Local edits respected via hash-checked lockfile.
terminal 
# 1. clone and bootstrap
$ mkdir my-venture && cd my-venture
$ mkdir .platform && cp ~/founderstaff/scaffolding/.platform/* .platform/

# 2. install core + the packs you want
$ python .platform/relearn.py --pack research --pack content

[relearn] cloning https://github.com/themeinnov8/founderstaff.git
[founderstaff] syncing 37 skills into my-venture
  install: chief-of-staff v0.1.0 -> .claude/agents/chief-of-staff.md
  install: market-research v0.1.0 -> .claude/commands/market-research.md
  ...

# 3. open Claude Code, instantiate your roster
$ claude
> /instantiate-agent chief-of-staff
> /schedule chief-of-staff daily at=09:00 prompt=/next-best-action
> /run-due
